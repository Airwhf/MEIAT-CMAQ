# Allocator

这里是一个用于存放空间分配因子的目录。

在运行[coarse_emission_2_fine_emission.py](../coarse_emission_2_fine_emission.py)以前，你需要下载这些数据用于空间分配。

# Download

**您可以到以下链接中选择性下载您所需要的数据。**
链接：https://pan.baidu.com/s/1g2-YYE3uuw24iWw_82dedA?pwd=whff

1. 土地利用类型数据
* 农田：agriculture.zip
* 工矿用地：industry.zip or power.zip

2. 人口密度数据
* 2017年人口密度数据集：landscan-global-2017_nodata.zip
* 2018年人口密度数据集：landscan-global-2018_nodata.zip

3. 道路数据
* 高速路：motorway.zip
* 主干道：primary.zip
* 二级干道：secondary.zip
* 社区道路：residential.zip
* 道路数据栅格化：roads-nodata-0.zip

4. 平均分配数据
* 平均分配数据：ave-nodata-0.zip
